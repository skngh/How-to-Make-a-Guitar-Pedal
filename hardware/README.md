# Hardware

This folder contains all hardware-related files used throughout the course, including the PCB design, enclosure artwork, and drill schematic.

---

## PCB (KiCad)

The KiCad project files for the custom PCB designed during the course can be found in this folder.

- Open the KiCad project to view the schematic and PCB layout.
- These files correspond to [this code here.](../code/ChorusPedal/)

## Enclosure Graphic (Adobe Illustrator)

The enclosure artwork used for UV printing is provided as an Adobe Illustrator file.

- This file is used to generate the UV print design for Tayda.
- This is mostly for example and reference if you wanna checkout how I structured everything!
